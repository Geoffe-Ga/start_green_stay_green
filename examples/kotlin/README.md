# Kotlin (Wear OS) Example: `wrist-counter`

A minimal example of the Kotlin project scaffold generated by:

```bash
start-green-stay-green init \
  --project-name wrist-counter \
  --language kotlin \
  --no-interactive
```

## Provenance

Every file under [`wrist-counter/`](wrist-counter/) is **verbatim generated
output**: the command above was run against this repository (with the AI
enhancement pass disabled, i.e. the deterministic Pass-1 scaffold) and
the Kotlin-specific files were copied here unmodified. Re-running the
command reproduces them byte-for-byte.

## What this example includes

| File | Generated by |
|------|--------------|
| `settings.gradle.kts` | Root project + `:app` module wiring (plus the no-wrapper disclosure note) |
| `build.gradle.kts` | Plugin versions тАФ AGP, Kotlin 2.0, Compose, Kover |
| `gradle.properties` | AndroidX opt-in, JVM settings |
| `app/build.gradle.kts` | Wear OS app module тАФ `jvmToolchain(17)`, Compose for Wear OS dependencies, Kover тЙе90% coverage bound (`koverVerifyDebug`) |
| `app/src/main/AndroidManifest.xml` | `wear` device profile (watch hardware feature + standalone app metadata) |
| `app/src/main/kotlin/com/example/wrist_counter/MainActivity.kt` | Jetpack Compose for Wear OS entry point |
| `app/src/test/kotlin/com/example/wrist_counter/GreetingTest.kt` | JUnit starter suite |
| `detekt.yml` | detekt config тАФ cyclomatic complexity тЙд10, potential-bugs rules |
| `.pre-commit-config.yaml` | Hooks: ktlint (format), detekt, gitleaks, detect-secrets, shellcheck, hygiene checks |
| `.github/workflows/ci.yml` | ubuntu-runner CI: quality job (ktlint, detekt, gitleaks, тЙе90% Kover coverage gate), JDK 17/21 test matrix, Wear OS debug-APK build тАФ CI provisions its own pinned Gradle |

The Gradle wrapper (`gradlew` + `gradle-wrapper.jar`) is **deliberately
absent**: binary artifacts are never scaffolded. Run `gradle wrapper`
once locally; the generated CI works without it.

## Prerequisites

- JDK 17+ (CI also exercises JDK 21) and a local Gradle install
- `brew install ktlint detekt` (or your platform's SDK manager) for the
  pre-commit hooks
- An Android SDK for building the Wear OS APK (Android Studio or
  command-line tools)

## Getting started with a generated project

```bash
gradle wrapper   # materialize gradlew once (not generated)
./gradlew build
pre-commit install
./scripts/check-all.sh
```
